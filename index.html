How to use Galaxy Organizer
  1. Create a new System with + New system.
  2. Add planets with + Planet.
  3. Use Switch to Edit to drag and modify planets.
  4. Edit a planet or the sun via the Editor panel on the right.
  5. Use Settings → Export/Import to back up or restore systems.
  6. Make sure to click the Switch to Link button when you are done!
  7. You can use Ctrl+Z to undo and Ctrl+Y to redo.
